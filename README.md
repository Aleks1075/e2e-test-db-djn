# DJ Nosework End-to-End Tests

This repository contains end-to-end (E2E) tests for the DJ Nosework application to ensure the quality and reliability of the frontend and backend integration.

## Project Overview

The E2E tests are built using Playwright, designed to simulate user interactions and verify the functionality of the DJ Nosework application across different browsers.

## Features

- User authentication tests
- Competition and training course registration tests
- Profile management tests
- Integration tests for frontend-backend interactions

## Technologies Used

- Playwright

## Getting Started

### Prerequisites

- Node.js
